私隱政策

數獨日和 — 最後更新：2026年7月1日

本私隱政策說明數獨日和（「本網站」）如何收集、使用及保護您的個人資料。

1. 資料收集

本網站為純前端靜態遊戲，不會主動收集任何個人資料。我們不使用 cookies、追蹤器或任何形式的數據存儲。

2. Google AdSense

本網站可能使用 Google AdSense 顯示廣告。Google 可能會使用 cookies 來投放個人化廣告。您可以前往 Google 廣告設定 管理您的廣告偏好。

Google 作為第三方供應商，會使用 cookies 來根據用戶過往瀏覽記錄投放廣告。Google 收集的資料包括但不限於：瀏覽器類別、裝置類型、大概位置及瀏覽行為。

3. 第三方服務

本網站托管於 GitHub Pages（GitHub, Inc.），其私隱政策可於 GitHub 私隱政策 查閱。

4. 資料安全

本網站使用 HTTPS 加密連線，確保您與網站之間的資料傳輸安全。

5. 聯絡我們

如有任何私隱相關問題，請透過 GitHub Issues 聯絡：github.com/mssallyct/sudoku-app
